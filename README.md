==>**** Overview****

This project demonstrates a complete end-to-end data analytics workflow.
It covers dataset loading, exploratory data analysis (EDA), data cleaning, SQL querying, dashboard creation in Power BI, and final reporting using Gamma.
The objective is to uncover insights, improve data quality, and build a clear analytical story supported by visuals and documentation.

==> **Dataset**

The dataset was provided as a CSV file and contains customer purchase information.

Key columns include demographics, purchase amounts, order details, product categories, and discount usage.

The dataset was analyzed to understand customer behavior, revenue patterns, and product performance.
